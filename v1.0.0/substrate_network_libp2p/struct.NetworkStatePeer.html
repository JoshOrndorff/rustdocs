pub struct NetworkStatePeer {
    pub endpoint: NetworkStatePeerEndpoint,
    pub version_string: Option<String>,
    pub latest_ping_time: Option<Duration>,
    pub enabled: bool,
    pub open: bool,
    pub known_addresses: HashSet<Multiaddr>,
}

Fields

endpoint: NetworkStatePeerEndpoint

How we are connected to the node.

version_string: Option<String>

Node information, as provided by the node itself. Can be empty if not known yet.

latest_ping_time: Option<Duration>

Latest ping duration with this node.

enabled: bool

If true, the peer is "enabled", which means that we try to open Substrate-related protocols with this peer. If false, we stick to Kademlia and/or other network-only protocols.

open: bool

If true, the peer is "open", which means that we have a Substrate-related protocol with this peer.

known_addresses: HashSet<Multiaddr>

List of addresses known for this node.
