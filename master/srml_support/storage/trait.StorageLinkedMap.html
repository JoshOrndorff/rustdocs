A strongly-typed linked map in storage.

Similar to StorageMap but allows to enumerate other elements and doesn't implement append.

fn decode_len<KeyArg: Borrow<K>>(key: KeyArg) -> Result<usize, &'static str> where
    V: DecodeLength + Len, 

Read the length of the value in a fast way, without decoding the entire value.

T is required to implement Codec::DecodeLength.

Note that 0 is returned as the default value if no encoded value exists at the given key. Therefore, this function cannot be used as a sign of existence. use the ::exists() function for this purpose.
