sr_primitives

[−][src]Module sr_primitives::weights

Primitives for transaction weighting.

Each dispatch function within decl_module! can have an optional #[weight = $x] attribute. $x can be any type that implements the ClassifyDispatch<T> and WeighData<T> traits. By default, All transactions are annotated with #[weight = SimpleDispatchInfo::default()].

Note that the decl_module macro cannot enforce this and will simply fail if an invalid struct (something that does not implement Weighable) is passed in.
