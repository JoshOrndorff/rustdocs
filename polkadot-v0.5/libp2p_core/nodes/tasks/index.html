libp2p_core::nodes

[−][src]Module libp2p_core::nodes::tasks

Management of tasks handling nodes.

The core type is a [task::Task], which implements futures::Future and connects and handles a node. A task receives and sends messages ([tasks::FromTaskMessage], [tasks::ToTaskMessage]) to the outside.

A set of tasks is managed by a [Manager] which creates tasks when a node should be connected to (cf. [Manager::add_reach_attempt]) or an existing connection to a node should be driven forward (cf. [Manager::add_connection]). Tasks can be referred to by [TaskId] and messages can be sent to individual tasks or all (cf. [Manager::start_broadcast]). Messages produces by tasks can be retrieved by polling the manager (cf. [Manager::poll]).
